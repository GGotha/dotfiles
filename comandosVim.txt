l => >

dw => apaga uma palavra
2dw => apaga duas palavras

p => insere a linha apagada

n => funciona apenas quando existem palavras marcadas, ex: no comando acima procumos por /console.log, se apertarmos n ele ira para os proximos console.logs
:noh => remove campos selecionados
